# X (Twitter)

## No live source

There is no usable winners library for X. Meta and Google both publish a transparency
library that covers political and issue ads with some creative visibility. X's own
transparency center is scoped to the EU only, and even inside that scope it frequently
omits the actual creative, showing spend and targeting metadata without the ad copy or
image. That gap means there is no reliable live pull for this channel. This file carries
a one-time distilled set of starter rules instead of a live source card, and it stays
that way until a usable public library appears.

## Starter rules

Distilled once from general hook-copy corpora (see `references/cross-channel.md` for the
source). No live X winners pull exists to validate these against.

1. Lead with a claim or a number in the first 7 words, X truncates and the first line is
   the whole pitch in most feed contexts.
2. Write in the voice of a person posting rather than a brand posting. First person,
   present tense, no exclamation-point energy.
3. One idea per post. A thread expands an idea after the hook lands, it does not carry
   the hook itself.
4. Ask a real question only when a real answer is wanted, rhetorical questions read as
   ad copy and get scrolled past.
5. Pair a short line of text with one image or one screenshot rather than a longer
   caption, image-plus-short-text reads native, long-text-plus-image reads like a flyer.
6. Avoid hashtags in the hook line, they read as spam on this platform specifically, even
   where they work elsewhere.
